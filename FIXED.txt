═══════════════════════════════════════════════════════════════

ШАГ 1: Проверка Python

Запустите: check_python.bat

Этот скрипт:
✓ Проверит наличие Python на компьютере
✓ Покажет версию и расположение
✓ Проверит pip
✓ Даст инструкции если Python не найден

Если Python не найден:
- Запустите check_python_run.bat для автоматической установки
- Или установите вручную с https://www.python.org/downloads/
  (ВАЖНО: отметьте "Add Python to PATH" при установке!)

═══════════════════════════════════════════════════════════════

ШАГ 2: Установка библиотек

Запустите: install.bat

Скрипт автоматически:
✓ Найдет Python (python, py или python3)
✓ Установит все необходимые библиотеки
✓ Покажет результат установки

═══════════════════════════════════════════════════════════════

ШАГ 3: Настройка webhook

1. Откройте setup_webhook.md
2. Следуйте инструкциям для создания webhook в Bitrix24
3. Скопируйте .env.example в .env:
   copy .env.example .env
4. Откройте .env и вставьте ваш webhook URL

═══════════════════════════════════════════════════════════════

ШАГ 4: Проверка подключения

Запустите: test_connection.bat

Должно появиться:
✓ Подключение успешно! Пользователь: Ваше Имя

Если ошибка:
- Проверьте URL в .env файле
- Убедитесь что webhook активен в Bitrix24
- Проверьте права доступа webhook (нужен CRM)

═══════════════════════════════════════════════════════════════

ШАГ 5: Использование

Запустите: menu.bat

Выберите нужный отчет:
1. Отчет по лидам
2. Отчет по сделкам
3. Отчет по контактам
4. Полный отчет CRM

Все отчеты сохраняются в папку reports/ в формате Excel

❓ ЧАСТЫЕ ПРОБЛЕМЫ:

1. "Python не найден"
   → Запустите check_python.bat

2. "pip не найден"
   → python -m ensurepip --upgrade

3. "Invalid webhook"
   → Проверьте URL в .env файле

4. "Access denied"
   → Добавьте права CRM в настройках webhook

5. "Module not found"
   → Запустите install.bat еще раз
