# Bitrix24 Intelligence Automation

Профессиональная система аналитики и контроля качества отделов продаж на базе Bitrix24 и искусственного интеллекта.

## 🧠 Функционал

- **Автоматический сбор данных**: Интеграция с Bitrix24 REST API для получения сделок, звонков и истории стадий.
- **ASR Pipeline**: Обработка аудио через Bit.Newton, VibeCode или локальные движки с поддержкой диаризации.
- **Conversation Intelligence**:
    - Глубокий скоринг звонков по кастомным чек-листам.
    - Детекция возражений и анализ их отработки.
    - Оценка эмоциональных рисков и уверенности менеджера.
- **CRM Hygiene**: Контроль заполнения полей, наличия следующего шага и корректности ведения воронки.
- **Reporting**: Генерация детальных Excel-отчетов и JSON-выгрузок для BI-систем.

## ⚙️ Технологический стек

- **Core**: Python 3.10+ (Asyncio).
- **Analysis**: Pandas, Pydantic, Regular Expressions.
- **Integrations**: Bitrix24 Webhooks/OAuth, Bit.Newton API, VibeCode API.
- **Interface**: Streamlit Web UI.

## 🚦 Быстрый старт

1. Настройте `.env` на основе `.env.example`.
2. Запустите процесс анализа: `.\run_full_report.bat`.
3. Откройте веб-интерфейс: `web_ui.py` через Streamlit.
