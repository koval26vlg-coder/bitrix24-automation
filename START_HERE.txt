🎯 ШАГ 1: УСТАНОВКА (5 минут)

   Запустите: install.bat

   Это установит все необходимые библиотеки Python


🔑 ШАГ 2: НАСТРОЙКА API (5 минут)

   1. Откройте файл: setup_webhook.md
   2. Следуйте инструкциям для создания webhook
   3. Скопируйте .env.example в .env
   4. Вставьте ваш webhook URL в .env


✅ ШАГ 3: ПРОВЕРКА

   Запустите: test_connection.bat

   Должно появиться: "✓ Подключение успешно!"


🚀 ШАГ 4: ИСПОЛЬЗОВАНИЕ

   Запустите: menu.bat

   Выберите нужный отчет из меню


📊 РЕЗУЛЬТАТ

   Все отчеты сохраняются в папке: reports/
   Формат: Excel (.xlsx)
