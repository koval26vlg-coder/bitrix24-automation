🔧 ЧТО СЛУЧИЛОСЬ:

Python не установлен или не добавлен в PATH


🚀 РЕШЕНИЕ (выберите один вариант):

═══════════════════════════════════════════════════════════════

ВАРИАНТ 1: Автоматическая проверка и установка (РЕКОМЕНДУЕТСЯ)

1. Запустите: check_python_run.bat

   Этот скрипт:
   ✓ Проверит наличие Python
   ✓ Предложит автоматическую установку
   ✓ Настроит PATH

2. После установки запустите: install.bat

═══════════════════════════════════════════════════════════════

ВАРИАНТ 2: Ручная установка Python

1. Откройте: https://www.python.org/downloads/

2. Скачайте последнюю версию (Python 3.12 или новее)

3. При установке ОБЯЗАТЕЛЬНО:
   ☑ Отметьте "Add Python to PATH"
   ☑ Отметьте "Install for all users" (опционально)

4. После установки перезапустите командную строку

5. Запустите: install.bat

═══════════════════════════════════════════════════════════════

ВАРИАНТ 3: Проверка существующей установки

Если Python уже установлен, но не найден:

1. Запустите: check_python.bat

   Скрипт покажет где искать Python

2. Найдите папку Python (обычно):
   C:\Python312\
   C:\Users\koval\AppData\Local\Programs\Python\Python312\

3. Добавьте в PATH:
   - Win+R → sysdm.cpl → Дополнительно → Переменные среды
   - Найдите PATH → Изменить
   - Добавить → вставьте путь к Python
   - Добавить → вставьте путь к Python\Scripts

4. Перезапустите командную строку

═══════════════════════════════════════════════════════════════

📝 ПОСЛЕ УСТАНОВКИ:

1. Откройте новое окно командной строки
2. Проверьте: python --version
3. Если работает, запустите: install.bat
4. Затем настройте webhook и запустите menu.bat
