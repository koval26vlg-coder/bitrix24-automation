🎯 СЛЕДУЮЩИЙ ШАГ: Настройка webhook

Вам нужно создать файл .env с вашим webhook URL от Bitrix24.

ВАРИАНТ 1: Быстрый (если у вас уже есть webhook)
---------------------------------------------------

Создайте файл .env в этой папке со следующим содержимым:

BITRIX24_WEBHOOK=https://online-kassa.bitrix24.ru/rest/USER_ID/WEBHOOK_CODE/

Замените USER_ID и WEBHOOK_CODE на ваши значения.


ВАРИАНТ 2: Подробный (если нужно создать webhook)
---------------------------------------------------

1. Откройте файл: setup_webhook.md
2. Следуйте пошаговой инструкции
3. Получите webhook URL
4. Создайте файл .env с этим URL

📝 КАК СОЗДАТЬ ФАЙЛ .env:

Способ 1 - Через Блокнот:
1. Откройте Блокнот
2. Вставьте: BITRIX24_WEBHOOK=ваш_webhook_url
3. Сохраните как: .env (с точкой в начале!)
4. Тип файла: Все файлы (не .txt!)

Способ 2 - Через командную строку:
1. Откройте командную строку в этой папке
2. Выполните: copy .env.example .env
3. Откройте .env и вставьте ваш webhook URL
