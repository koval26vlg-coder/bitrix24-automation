# Инструкция по настройке Webhook для Bitrix24

## Шаг 1: Войдите в Bitrix24

Откройте ваш портал: https://online-kassa.bitrix24.ru

## Шаг 2: Создание входящего вебхука

1. В левом меню найдите **"Приложения"** или **"Разработчикам"**
2. Выберите **"Вебхуки"** → **"Входящий вебхук"**
3. Нажмите **"Добавить вебхук"**

## Шаг 3: Настройка прав доступа

Выберите необходимые права для вебхука:

### Обязательные права:
- ✅ **CRM** - чтение и запись
  - crm (все операции с CRM)
- ✅ **Диск** - чтение
  - disk (доступ к файлам)
- ✅ **Пользователи** - чтение
  - user (информация о пользователях)
- ✅ **Телефония** - чтение/запись транскрипций
  - telephony / voximplant

## Шаг 4: Получение URL

После создания вебхука вы получите URL вида:

```
https://online-kassa.bitrix24.ru/rest/123/a1b2c3d4e5f6g7h8/
```

Где:
- `123` - ID пользователя
- `a1b2c3d4e5f6g7h8` - секретный код вебхука

## Шаг 5: Сохранение в .env

1. Скопируйте файл `.env.example` в `.env`:
   ```bash
   cp .env.example .env
   ```

2. Откройте `.env` и вставьте ваш URL:
   ```
   BITRIX24_WEBHOOK=https://online-kassa.bitrix24.ru/rest/123/a1b2c3d4e5f6g7h8/
   BITRIX24_DOMAIN=online-kassa.bitrix24.ru
   BITNEWTON_ASR_URL=https://bit-asr.1bitai.ru
   BITNEWTON_TOKEN=your_bitnewton_token
   ```

3. Сохраните файл

## Шаг 6: Проверка подключения

Запустите тестовый скрипт:

```bash
python -c "from bitrix24_api import Bitrix24API; api = Bitrix24API(); api.test_connection()"
```

Если все настроено правильно, вы увидите:
```
✓ Подключение успешно! Пользователь: Ваше Имя
```

## Безопасность

⚠️ **ВАЖНО:**
- Никогда не публикуйте URL вебхука в открытых источниках
- Не коммитьте файл `.env` в git (он уже в `.gitignore`)
- Регулярно обновляйте вебхуки
- Используйте минимально необходимые права доступа

## Альтернатива: OAuth приложение

Для продакшена рекомендуется использовать OAuth:

1. Перейдите в **"Разработчикам"** → **"Другое"** → **"Локальное приложение"**
2. Создайте приложение
3. Получите CLIENT_ID и CLIENT_SECRET
4. Реализуйте OAuth flow (требуется дополнительный код)

## Troubleshooting

### Ошибка: "Invalid webhook"
- Проверьте правильность URL
- Убедитесь, что вебхук не был удален в Bitrix24

### Ошибка: "Access denied"
- Проверьте права доступа вебхука
- Добавьте необходимые права в настройках вебхука

### Ошибка: "Connection timeout"
- Проверьте интернет-соединение
- Убедитесь, что портал доступен
